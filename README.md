# Typing Simulator (C)
All the fun of typing, now in C!  
itch.io page: https://missingpro.itch.io/typingsimulator-c
## Compiling from source
Because all of GitHub has this.  
The source code is typingsimulator.c - I build it using MSYS2's version of MINGW64, and in there I run gcc -o typingsimulator.exe typingsimulator.c - that's it!
